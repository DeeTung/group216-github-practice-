## 1. Giới thiệu project

Đây là repository thực hành Git & GitHub của nhóm 216.

Project được xây dựng bằng ngôn ngữ Python với mục tiêu thực hành quy trình làm việc nhóm trên GitHub, bao gồm:

- Tạo repository
- Tạo branch riêng cho từng thành viên
- Commit code
- Push code lên GitHub
- Tạo Pull Request
- Review và merge Pull Request
- Xử lý conflict cơ bản

Project mẫu của nhóm là chương trình **Quản lý sinh viên**.
